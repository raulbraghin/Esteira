# Esteira
Esqueira empilhadora de pallets

Aqui é o progeama de uma maquina piloto empilhadora de palets

Escrito em C++ na Arduino IDE com o Arduino Mega, motores de passo, LCD 16x2 I2C, botões e sensore on/off
